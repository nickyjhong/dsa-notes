## Vocabulary
- **Head** - beginning of linked list
- **Tail** - end of linked list
- **Length** - how long the linked list is
- **Node** - an element that references next node (or if tail, references null)
- **Value** 
- **Pointer** - single or double pointer to next node
  - **Next**
  - **Previous**
